Segue por completo o Sistema de Controle de Frota v2.

Abaixo estão as explicações sobre cada arquivo da pasta:

*Executavel: Dentro desta pasta está um arquivo .jar que pode ser executado via linha de comando, navegando até a pasta e utilizando o código java -jar "SCFApplication.jar", e uma pasta "lib" contendo as bibliotecas necessárias para o projeto
 
*Diagrama: Digrama de classes em formato .Jpeg

*Diagrama em PDF: Diagrama de classes em formato .pdf

*Dump20231207 : Banco de dados da aplicação com algumas inserções de exemplo

*Proposta do Projeto: Arquivo pdf com a explicação da proposta do projeto

*SCF: Arquivo em formato .zip para importação em IDE com suporte a  linguagem Java com os códigos da aplicação

*Script_de_criacao: Caso tenha alguma dificuldade com a importação do banco de dados, segue um script de criação do banco de dados, com comandos úteis.
